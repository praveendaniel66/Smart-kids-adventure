# Smart Kids Adventure – Educational Desktop Game

## Project Overview

**Smart Kids Adventure** is a simple educational desktop game designed for children aged 3–8 years.
The game helps kids learn basic concepts such as alphabets, numbers, colors, animals, fruits, and simple words through interactive gameplay.

This project is developed as a **desktop application using GDevelop**, with simple keyboard controls and colorful visual elements to make learning fun and engaging.

---

## Features

* 2D educational game designed for kids
* Desktop-based gameplay
* Simple **keyboard controls (Arrow Keys)**
* Colorful and child-friendly interface
* Multiple levels with different learning topics
* Word collection runner mode
* Random word generation
* Score and level progression system

---

## Game Modes

### 1. Learning Levels Mode

This mode contains **10 levels** where players move the character and select the correct answer.

Example levels include:

* Alphabet recognition
* Number identification
* Color recognition
* Animal identification
* Shape recognition
* Simple math questions
* Fruit identification
* Opposite words
* Nature objects
* Final mixed quiz

The player must move the character to collide with the **correct answer** to complete each level.

---

### 2. Word Runner Mode

This mode is inspired by **endless runner games**.

Features:

* Player runs automatically
* Player must collect correct words
* Obstacles appear randomly
* If the player hits an obstacle, the game ends
* The player must collect all required words to complete the level

This mode contains **5 levels**, and each time the game starts the words are **randomly generated**.

## How to Run the Game

1. Download or clone the project.
2. Open the project using **GDevelop**.
3. Click **Preview** to test the game.
4. Export the game as a **Windows Desktop (.exe)** file.
